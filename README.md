# exif-edit

一款基于 vue 的图片 exif 数据编辑器组件。

An exif data edtior component with javascript(based on vue).

<p align="center">
  <img src="./docs/imgs/7.jpg" alt="7.jpg" width="400" />
</p>

## 安装依赖
```
yarn install
```

### 开发环境启动
```
yarn serve
```

### 构建打包 demo 应用
```
yarn build:app
```

### 构建打包组件
```
yarn build:lib
```

### 构建打包 web-component
```
yarn build:wc
```

### 代码检查
```
yarn lint
```
